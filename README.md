# 图片压缩网站

## 项目简介
这是一个功能完善、设计精美的图片压缩网站，支持用户上传PNG、JPG等格式的图片，按比例进行压缩，减少图片文件大小。

## 功能特性
1. **图片上传**：支持PNG、JPG、JPEG等常见图片格式
2. **比例压缩**：用户可自定义压缩质量（0-100%）
3. **实时预览**：同时显示上传的原始图片和压缩后的图片
4. **文件大小对比**：清晰展示压缩前后的文件大小变化
5. **图片下载**：一键下载压缩后的图片

## 页面结构说明

### 布局结构
- **头部区域**：网站标题和简介
- **上传区域**：拖拽上传或点击上传图片
- **控制面板**：压缩质量调节滑块
- **预览区域**：左右并排显示原图和压缩后的图片
- **信息显示**：文件大小对比和压缩比例
- **下载按钮**：下载压缩后的图片

### 样式设计
- **设计风格**：采用苹果风格的简约设计
- **配色方案**：以白色和浅灰色为主，配合柔和的蓝色和阴影效果
- **响应式布局**：使用Flexbox和Grid布局，适配不同屏幕尺寸
- **交互动效**：平滑的过渡动画和悬停效果

## 技术实现
- **HTML5**：语义化标签，结构清晰
- **CSS3**：Flexbox和Grid布局，响应式设计
- **JavaScript**：使用Canvas API进行图片压缩处理
- **无框架依赖**：纯原生实现，性能优异

## 使用说明
1. 打开 `index.html` 文件即可使用
2. 点击上传区域或拖拽图片到指定区域
3. 调整压缩质量滑块（0-100%）
4. 实时预览压缩效果
5. 点击下载按钮保存压缩后的图片
